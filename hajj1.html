Hajj & Flights Analysis

Interactive Dashboard & Data Analysis Project

📊 Project Overview

This project analyzes the relationship between the number of pilgrims and flights in Saudi Arabia. The goal is to identify trends, seasonal patterns, and capacity planning insights.

🛠 Tools Used

Power BI, Python, Excel

📈 Key Insights

  • Increase in pilgrims leads to higher flight frequency
  • Clear seasonal peaks during Hajj period
  • Opportunities for optimizing airline scheduling
