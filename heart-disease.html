Project Overview

This project analyzes a real heart disease dataset from Kaggle containing 920 patient records from multiple countries (Cleveland, Hungary, Switzerland, VA Long Beach). The goal was to explore the data, understand which factors are linked to heart disease, and build a machine learning model that can predict whether a patient has heart disease based on their clinical data.

Project Pipeline
01
Data Exploration
Loaded the CSV dataset using Pandas, inspected the first rows, data types, and identified missing values across 17 columns.
02
Data Visualization
Used Seaborn and Matplotlib to create charts showing disease distribution by age groups, gender, chest pain type, blood pressure, and cholesterol.
03
Data Cleaning & Feature Engineering
Filled missing values using SimpleImputer, created age category columns (<40, 40-55, 55-70, 70+), and built interaction features to improve model performance.
04
Model Building
Applied Logistic Regression using Scikit-learn Pipeline with OneHotEncoder for categorical variables and StandardScaler for numerical ones. Split data 80/20 for training and testing.
05
Model Evaluation
Evaluated using accuracy, precision, recall, F1-score, and confusion matrix — achieving 98% overall accuracy.
Key Findings
  • 01 Age is a strong factor — heart disease rates rise significantly in the 55–70 age group, with the highest incidence recorded there.
  • 02 Gender differences exist — males showed higher rates of heart disease compared to females in the dataset.
  • 03 High blood pressure and elevated cholesterol were more common among patients diagnosed with heart disease.
  • 04 Asymptomatic chest pain type showed the strongest association with heart disease presence among all chest pain categories.
  • 05 Decreased performance during exercise (oldpeak) correlated with higher risk of heart disease.
